流程的概念与分类
	概念：程序步骤执行的先后顺序
	分类：
		1、顺序结构：从上到下，从前到后，顺序执行
		2、选择结构：执行路线分叉（分支结构）
			单if语句：if（布尔表达式）{
                                                                    语句体                      
				}
			标准的if-else语句
				if（布尔表达式）{
				       语句体A
				}else{
				       语句体B	
				}
			扩展<Ext>的if-else语句
				if(条件判断1){
                                                                  语句体1		
				}else if（条件判断2）{
				      语句体2
				}...else{
				       语句体N+1
				}
		3、循环结构：重复做一些事情
			1、初始化语句（准备工作），只执行一次
			2、条件判断（布尔结果），如果不够，就继续写，如果够了，就停下来，每次动笔停下来都判断
			3、循环体，工作内容，需要重复做什么
			4、收尾工作，每次执行完都执行一次步进语句

for循环的格式与流程
	格式：for（1初始化语句(执行唯一一次)，2条件判断，4步进语句）{
		3循环体
		} 
执行顺序1234=>234...
while循环
	标准格式：while（条件判断）{
			循环体
			}
	扩展格式：1初始化语句
		while（2条件判断）{
			3循环体
			4步进语句
			}
for循环与while循环的区别：
	1、for循环格式固定，while循环格式灵活
	2、while循环的初始化表达式在外面
跳转控制语句（关键字）
	1、break语句//打断循环
	2、continue语句//结束当前的这次循环，继续下一次循环
死循环（永真循环）
		标准格式：while(true){
			循环体
			}
		扩展格式：（；/*true*/；）{
			循环体
			}
循环嵌套中的控制语句
	break和continue只能默认作用于所属层次的循环
	如果希望写在内层作用在外层，使用标签名称   break 标签名称
循环嵌套:外层循环每执行一次，都对应着内层循环执行多次。
